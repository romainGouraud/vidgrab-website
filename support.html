Common questions

How do I install VidGrab? +
  1. Open the Chrome Web Store listing (link coming once we publish).
  2. Click Add to Chrome.
  3. Confirm the permissions (we need them to detect videos on web pages).
  4. Pin the VidGrab icon to your toolbar.

You'll automatically get 3 days of unlimited downloads to test all features.

I purchased Pro but the badge still says Free +

Try this in order:

  1. Close and reopen the popup - license is re-checked on each open.
  2. Open the Options page → Premium tab → click Already paid? Restore your license.
  3. Enter your purchase email when prompted by ExtensionPay.
  4. If still not working, email support@vidgrab.app with your purchase email and we'll fix it manually within minutes.
VidGrab doesn't detect any video on a page +

Try the following:

  • Click the play button on the video first. Many platforms only fetch the video URL once playback starts. VidGrab needs to see the network request to detect it.
  • Click the Refresh scan button (circular arrow icon) in the popup header.
  • Reload the page (F5) and re-open the popup.
  • If the video is on YouTube, it won't work - Chrome Web Store policy blocks YouTube downloaders.
  • If the video is DRM-protected (Netflix, Disney+, paid streaming), it cannot be downloaded by any tool.
The download fails or stops at 95% +

This usually means the source CDN rejected one or more segments. Try:

  • Make sure the video is still playing in the original tab when you start the download.
  • Click Retry on the failed video card.
  • For HLS streams, sometimes the lower-quality variant works when the highest fails - pick a lower quality from the dropdown.
  • If consistent failure on a specific site, email us with the URL - we'll investigate.
How do I change the extension's language? +

Right-click the VidGrab icon → Options → Settings tab → Language dropdown. Choose between Auto (browser), English, Español, Português (BR), Français, Deutsch.

I want a refund +

30-day money back, no questions asked. Email support@vidgrab.app from your purchase email. Full details on our Refund Policy page.

Can I use VidGrab on Edge / Brave / Opera? +

Yes - any Chromium-based browser supports Chrome Web Store extensions. Microsoft Edge, Brave, Opera, Vivaldi all work. Just install from the Chrome Web Store.

How do I uninstall? +

Right-click the VidGrab icon in your toolbar → Remove from Chrome… → confirm. All local data (settings, history) is wiped automatically. If you've purchased Pro, your license remains valid - you can reinstall anytime and restore it via your purchase email.
